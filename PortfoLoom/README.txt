# PortfoLoom

## Overview

Welcome to **PortfoLoom**! This project is a resume and portfolio generator website built using HTML, CSS, and JavaScript. 
The website allows users to create professional resumes and portfolios with ease. 
The entry point of the project is `hellopage.html` located in the `helloPage` folder. 
From this page, you can navigate to subsequent pages or access other parts of the website.

## Prerequisites

To run this project, you need:

- A web browser (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge, Safari)

## Getting Started

Follow these steps to run the project:

1. Download or Clone the Repository

   You can download the project as a ZIP file and extract it, or clone the repository using Git.

   ```bash
   git clone https://github.com/wivinwinny8/PortfoLoom-Resume_and_Portfolio_builder.git
   ```

2. Navigate to the Project Directory

   Open your terminal or file explorer and navigate to the directory where you downloaded or cloned the project.

3. Open the Entry Point

   Navigate to the `helloPage` folder and open the `hellopage.html` file in your web browser. 
   You can do this by double-clicking the file or right-clicking and selecting "Open with" followed by your preferred web browser.

   ```bash
   cd PortfoLoom-Resume_and_Portfolio_builder/PortfoLoom
   cd helloPage
   start hellopage.html
   ```

4. **Browse the Website**

   Once `hellopage.html` is open in your web browser, you can navigate through the website using the links provided on the page. 
   You can also directly access other pages by typing their URLs in the browsers address bar if you know the paths.

## Project Components

- helloPage folder : Contains the entry point `hellopage.html`, styles, and images for the home page.
- forms folder : Contains form pages (`form1.html`, `form2.html`), their styles (`form1style.css`, `form2style.css`), a script file (`script1.html`), and template images.
- final folder : Contains final pages (`final1.html`, `final2.html`) and their styles (`final1style.css`, `final2style.css`).
- portfolio folder : Contains portfolio pages (`portfolio1.html`, `portfolio2.html`), their styles (`portfolio1style.css`, `portfolio2style.css`), and images (`github.png`, `user.png`).
- resumeTemplates folder : Contains resume pages (`resume1.html`, `resume2.html`), their styles (`resume1style.css`, `resume2style.css`), and images (`git.png`, `linkedin.png`).
- signinPage folder : Contains the sign-in page (`signin.html`) and its styles (`signinstyle.css`).
- themeSelection folder : Contains the theme selection page (`selection.html`), its styles (`selectionstyle.css`), and template images.

## JavaScript Usage

JavaScript is used to provide interactive features and dynamic content generation for both the resume and portfolio pages. 
Make sure your browser has JavaScript enabled to fully experience the functionality of the website.

## Troubleshooting

If you encounter any issues:

- Ensure all files are in their correct directories.
- Verify that your web browser is up to date.
- Check the console in your web browsers developer tools for any error messages that might help diagnose the problem.
- Ensure JavaScript is enabled in your browser settings.
